# russian-sudo
Makes bash understand russian

# installation:

```bash
curl https://raw.githubusercontent.com/dimagoltsman/russian-sudo/main/install | bash
```
##or if you use zsh
```bash
curl https://raw.githubusercontent.com/dimagoltsman/russian-sudo/main/install | zsh
```
